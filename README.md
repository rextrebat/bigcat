bigcat
======


Benchmark
---------

A *benchmark* is the basic unit of configuration of bigcat. It encapsulates the workload to be executed along with the driver infrastructure required to execute it. 

A *benchmark* may be modeled as a simple set of *operations* or, to may be modeled to mimic user sessions using the *session* entity. *Sessions* may also be used to mimic a conceptual sequence of transactions even if there is no user session involved.
